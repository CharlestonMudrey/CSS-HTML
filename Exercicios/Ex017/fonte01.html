Alinhamento de textos usando CSS

Este primeiro parágrafo terá o alinhamento padrão, à esquerda.

O segundo parágrafo terá alinhamento à direita.

O terceiro parágrafo terá alinhamento centralizado.

O último parágrafo terá o alinhamento justificado. Para que você perceba isso, terei que colocar um texto um pouco maior, caso você não perceba que o alinhamento está sendo feito nas duas laterais, tente diminuir o tamanho do seu navegador. Este tipo de alinhamento é bastante usado para textos longos e grandes parágrafos, para dar uma sensação de simetria. Agora o que escrevo é para encher linguiça.
